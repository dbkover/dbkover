DBKover - Database testing made easy

DBKover is a library to enable easy integration testing to databases using DBUnit.

Get Started

Open-source Apache-2.0 Licensed. GitHub v0.2.1

⚡️ Fast test development

Write your database integration tests quickly by leveraging DBKovers easy setup and maintenance.

✅ Assertable database changes

Asserting database manipulations has never been easier. Provide DBKover a data set of how the db should look like and watch the test result.

⚙️ Framework integrations

DBKover is integrated directly into JUnit 5 lifecyle and more frameworks is on the roadmap.
